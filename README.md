# Описание 1 задания

Программа проверяет корректность данных, вычисляет среднюю оценку и список учащихся, прошедших экзамен.

## Входные данные

Программа принимает данные в виде списка строк, где каждая строка содержит фамилию ученика и его оценку, разделенные двоеточием. Пример строки:

```
Иванов: 85
```

## Функции программы

1. **process_grades(records: list[str]) -> dict**

   Основная функция для обработки данных.

   Возвращает словарь с результатами обработки:

     `valid_count`: количество валидных оценок.

     `average`: средняя оценка.

     `passed`: список учащихся, прошедших экзамен (отсортирован по фамилии).

     `skipped`: количество пропущенных записей.

2. **read_data() -> list[str]**

   Функция для чтения данных из файла.

   Возвращает список строк в формате ```Иванов: 85```

Если данные передаются напрямую или из другой функции, удалите блок с чтением файла


# Описание 2 задания

Программа ищет самую длинную растущую серию чисел

## Функции программы

1. **longest_increasing_streak(nums: list[int]) -> dict**

   Основная функция для обработки данных.

   Возвращает длину самой длинной цепочки чисел подряд по возрастанию и саму цепочку
